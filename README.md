Here is a cheat sheet for both JavaScript and Ruby. Still a work in progress.

# JavaScript

## concat_str.js

Shows the different methods to put strings together.

- .concat()
- "+"
- "+="
- .join()
- .repeat()

## str_manipulation.js

Shows the different methods to manipulate strings.

- .toLowerCase()
- .toUpperCase()
- .trim()
- .trimLeft()
- .trimRight()
